# Moly Kit

## 简介

Moly Kit 是一个 Rust crate，包含组件与工具，用于简化 Makepad 框架中的 AI 应用开发。

它提供了一系列出色的新特性，为开发 Makepad 应用时的组件设计提供必要参考，同时加速你的开发与使用体验。

- ⚡️ 低配置聊天组件，几乎开箱即用。
- 🔧 可自定义并集成 Chat 行为到你的应用中。
- 🎨 通过 Makepad DSL 覆写实现外观定制。
- 📞 内置兼容 OpenAI 的客户端。
- 🧩 可扩展自定义客户端与消息内容。
🌎 支持 Web。
- 如需了解如何在自己的 Makepad 应用中使用并集成 Moly Kit，请阅读文档。

要了解如何在自己的 Makepad 应用中使用并集成 Moly Kit，请阅读文档：https://moly-ai.github.io/moly-ai.
