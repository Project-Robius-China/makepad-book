# exp2

return 2 raised to the power of the parameter

## Declaration
- ``genType exp2(genType x)``
## Parameters
- ``x``:  Specify the value of the power to which 2 will be raised.
## Description
`exp2` returns 2 raised to the power of _`x`_. i.e.,
$$
2^{x}
$$
