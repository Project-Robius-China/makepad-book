# General API

## app_main!(App);

The `app_main!` macro is the entry point for your application. This struct must implement the `AppMain` trait.

## live_design!\{Live DSL code\}

The `live_design!` macro is used to define a live design code, which is a DSL for defining UIs.

More Live DSL code details can be found in the [Live DSL](/guide/live-DSL/index) guide.
